# Python Strings Assessment

**Topic Covered:** Variables, Assignment, String Concatenation, f-Strings, String Methods, Indexing, and Slicing

Assign one section to each trainee. Each section contains **4 progressively harder exercises**.

### Exercise 2


* the original text
* uppercase version

* title case version
